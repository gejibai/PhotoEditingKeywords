# 日常修图关键词完善器

Next.js App Router + TypeScript + Tailwind CSS MVP for converting rough daily photo editing ideas into structured prompts.

## Run

```bash
pnpm install
pnpm dev
```

## Environment

Server-only AI key:

```bash
OPENAI_API_KEY=your_api_key_here
```

Do not use `NEXT_PUBLIC_OPENAI_API_KEY`.
